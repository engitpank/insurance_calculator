**Контекст задачи (краткое описание) **
Заполнить поля ответа (TravelCalculatePremiumResponse), значения для заполнения взять из запроса (TravelCalculatePremiumRequest).
Формирование ответа это часть бизнес логики приложения, поэтому требуемые изменения внесите в класс TravelCalculatePremiumServiceImpl.

**Что было сделано и как? (краткое описание)**
В методе calculatePremium класса TravelCalculatePremiumServiceImpl в возвращаемом объекте был вызван конструктор
для TravelCalculatePremiumResponse, в который передали все поля из полученного запроса.
